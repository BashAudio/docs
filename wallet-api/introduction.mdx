The Mufi Wallet API is a RESTful API that allows you to interact with the Mufi Wallet. It enables you to easily create custodial wallets on the Polkadot network paired with user email addresses.
These wallets are secure, easy to use, and allow you to interact with web3 applications and services in a web2 flow. 

Simply follow the steps below to enable your project to support these wallets in your project.

## Overview

### Authentication 

All API endpoints are authenticated using Bearer tokens received from the [Get Auth Token](/wallet-api/endpoint/auth) endpoint using your JWT credentials.

<CodeGroup>
  ```bash cURL
  curl -H "Authorization: Bearer $YOUR_ACCESS_TOKEN" https://wallets-api-staging-948987300951.us-east4.run.app/your-endpoint
  ```

  ```python Python
  import os
  import requests

  token = os.getenv('YOUR_ACCESS_TOKEN')
  headers = {'Authorization': f'Bearer {token}'}

  response = requests.get('https://wallets-api-staging-948987300951.us-east4.run.app/your-endpoint', headers=headers)
  print(response.json())
  ```

  ```javascript JavaScript
  const fetch = require('node-fetch');

  const token = process.env.YOUR_ACCESS_TOKEN;

  const response = await fetch('https://wallets-api-staging-948987300951.us-east4.run.app/your-endpoint', {
      headers: {
          'Authorization': `Bearer ${token}`
      }
  });

  const data = await response.json();
  console.log(data);
  ```

  ```php PHP
  <?php
  $token = getenv('YOUR_ACCESS_TOKEN');
  $url = 'https://wallets-api-staging-948987300951.us-east4.run.app/your-endpoint';
  $options = [
      'http' => [
          'header' => "Authorization: Bearer $token"
      ]
  ];
  $context = stream_context_create($options);
  $result = file_get_contents($url, false, $context);
  echo $result;
  ?>
  ```

  ```go Go
  package main

  import (
      "fmt"
      "net/http"
      "io/ioutil"
      "os"
  )

  func main() {
      token := os.Getenv("YOUR_ACCESS_TOKEN")
      client := &http.Client{}
      req, _ := http.NewRequest("GET", "https://wallets-api-staging-948987300951.us-east4.run.app/your-endpoint", nil)
      req.Header.Add("Authorization", "Bearer " + token)

      resp, _ := client.Do(req)
      body, _ := ioutil.ReadAll(resp.Body)
      fmt.Println(string(body))
  }
  ```

  ```java Java
  import java.io.IOException;
  import java.net.HttpURLConnection;
  import java.net.URL;
  import java.util.Scanner;

  public class Main {
      public static void main(String[] args) throws IOException {
          String token = System.getenv("YOUR_ACCESS_TOKEN");

          URL url = new URL("https://wallets-api-staging-948987300951.us-east4.run.app/your-endpoint");
          HttpURLConnection conn = (HttpURLConnection) url.openConnection();
          conn.setRequestMethod("GET");
          conn.setRequestProperty("Authorization", "Bearer " + token);

          Scanner scanner = new Scanner(conn.getInputStream());
          while (scanner.hasNext()) {
              System.out.println(scanner.nextLine());
          }
          scanner.close();
      }
  }
  ```
</CodeGroup>
