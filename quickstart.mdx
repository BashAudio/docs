## 2. Setup in Your Project

To keep your credentials safe and out of sight, it's best to store them in environment variables. This way, your credentails stay hidden and secure, 
and you don't have to worry about it showing up in your code. Plus, it's super easy to manage different sets of credentials for different environments like development, staging, and production.

### Use your Credentials in your project

<CodeGroup>
  ```bash cURL
  curl -u $MUFI_API_USERNAME:$MUFI_API_PASSWORD https://wallets-api-staging-948987300951.us-east4.run.app/auth/token
  ```

  ```python Python
  import os
  import requests
  from requests.auth import HTTPBasicAuth

  username = os.getenv('MUFI_API_USERNAME')
  password = os.getenv('MUFI_API_PASSWORD')

  response = requests.get('https://wallets-api-staging-948987300951.us-east4.run.app/auth/token', auth=HTTPBasicAuth(username, password))
  print(response.json())
  ```

  ```javascript JavaScript
  const fetch = require('node-fetch');

  const username = process.env.MUFI_API_USERNAME;

  const response = await fetch('https://wallets-api-staging-948987300951.us-east4.run.app/auth/token', {
      headers: {

      }
  });

  const data = await response.json();
  console.log(data);
  ```

  ```php PHP

  $username = getenv('MUFI_API_USERNAME');
  ```

  ```go Go
  package main

  import (
      "fmt"
      "net/http"
      "io/ioutil"
      "os"
      "encoding/base64"
  )

  func main() {
      username := os.Getenv("MUFI_API_USERNAME")
      password := os.Getenv("MUFI_API_PASSWORD")
      client := &http.Client{}
      req, _ := http.NewRequest("GET", "https://wallets-api-staging-948987300951.us-east4.run.app/auth/token", nil)
      auth := base64.StdEncoding.EncodeToString([]byte(username + ":" + password))
      req.Header.Add("Authorization", "Basic " + auth)

      resp, _ := client.Do(req)
      body, _ := ioutil.ReadAll(resp.Body)
      fmt.Println(string(body))
  }
  ```

  ```java Java
  import java.io.IOException;
  import java.net.HttpURLConnection;
  import java.net.URL;
  import java.util.Base64;
  import java.util.Scanner;

  public class Main {
      public static void main(String[] args) throws IOException {
          String username = System.getenv("MUFI_API_USERNAME");
          String password = System.getenv("MUFI_API_PASSWORD");
          String auth = Base64.getEncoder().encodeToString((username + ":" + password).getBytes());

          URL url = new URL("https://wallets-api-staging-948987300951.us-east4.run.app/auth/token");
          HttpURLConnection conn = (HttpURLConnection) url.openConnection();
          conn.setRequestMethod("GET");
          conn.setRequestProperty("Authorization", "Basic " + auth);

          Scanner scanner = new Scanner(conn.getInputStream());
          while (scanner.hasNext()) {
              System.out.println(scanner.nextLine());
          }
          scanner.close();
      }
  }
  ```

### Credential Security Tips
- Make sure to always use HTTPS when making requests to the Mufi API to keep your data encrypted and secure.
- Never share your credentials with anyone.
- Use a password manager to store your credentials securely.
- Use environment variables to store your credentials in your project.
